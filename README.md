# Ascaris and Parascaris Hi-C Analysis

Code and reference files for
[Simmons et al. (2026)](https://www.biorxiv.org/content/10.1101/2025.10.23.684251v2)
*Spatial genome organization in nematodes with programmed DNA elimination*. bioRxiv.

Hi-C analysis of chromatin interactions during programmed DNA elimination in
*Ascaris suum* and *Parascaris univalens*.

## Data availability

Raw sequencing reads are deposited in GEO under accessions
[GSE314626](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE314626)
(Ascaris) and
[GSE315650](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE315650)
(Parascaris).

Hi-C matrices used by the scripts in this repository are ICE-normalized sparse
matrices generated by [HiC-Pro](https://github.com/nservant/HiC-Pro). To
regenerate them, run HiC-Pro with the appropriate reference genome (*Ascaris*
v50 or *Parascaris* v2) and apply ICE balancing. Eigenvector files (derived
from Hi-C matrices using [FAN-C](https://fan-c.readthedocs.io/) `fanc
compartments`) are included in this repository under `data/eigenvectors/`.

parameter descriptions, and a figure mapping table.

## Analysis descriptions

| Directory | Script | Description |
|-----------|--------|-------------|
| `interaction_scoring` | `plot_interaction_heatmap.py` | Quantify Hi-C interaction frequency between defined genomic region pairs |
| `hic_triangle_plots` | `plot_hic_stages.py` | Triangular Hi-C contact heatmaps across developmental timepoints |
| `ascarid_eigenvector_comparison` | `plot_ev1_multirow.py` | Side-by-side *Ascaris* vs *Parascaris* eigenvector 1 comparison |
| `developmental_eigenvector_chromosome_stacked` | `plot_ev1_single.py` | *Ascaris* EV1 stacked across developmental stages |
| `saddle_plot_analysis` | `saddle_plots.py` | Compartment saddle plots and compartment strength quantification |
| `multiomics_ab_compartments` | `multiomics_ab_compartments.py` | Compare A/B compartment signal across 8 omics datasets |
| `clustering_heatmap` | `clustering_heatmap.py` | Hierarchical clustering of Hi-C interaction patterns |
| `distance_decay` | `distance_decay.py` | Distance decay curves with three subcommands: `run`, `combine`, `plot` |
| `somatic_chromosome_end_analysis` | `analyze_ends_timepoints.py` | Multi-omics signal at new chromosome ends vs internal break regions |
| `plot_insulation` | `plot_insulation.py` | Per-chromosome insulation score plots with comprehensive boxplot |
| `insulation_genome` | `plot_insulation_genome.py` | Genome-wide stacked insulation scores across all chromosomes |
| `eigenvector_genome` | `plot_eigenvector_genome.py` | Genome-wide stacked EV1 across all chromosomes |
| `trans_cbr_profile` | `trans_cbr_profile.py` | Trans Hi-C interaction profiles and chromosome component breakdown at CBRs |
| `cbr_heatmap` | `cbr_heatmap.py` | CBR-by-CBR contact heatmap with PCA and trans interaction statistics |
| `hic_subregion` | `plot_hic_subregion.py` | Custom subregion Hi-C contact heatmaps for any coordinate range |

## Dependencies

All scripts are Python 3 and use standard scientific packages:

```
numpy
pandas
matplotlib
seaborn
scipy
```

Additional tools used upstream (not required to run these scripts):
- [HiC-Pro](https://github.com/nservant/HiC-Pro) — Hi-C read mapping, matrix generation, ICE normalization
- [FAN-C](https://fan-c.readthedocs.io/) — eigenvectors, insulation scores, expected values
