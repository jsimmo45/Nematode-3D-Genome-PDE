# Ascaris and Parascaris Hi-C Analysis

Code and reference files for [Simmons et al. (2026) Journal TBD].

Hi-C analysis of chromatin interactions during programmed DNA elimination in
*Ascaris suum* and *Parascaris univalens*.

## Data availability

Raw Hi-C sequencing reads are deposited in SRA under accession SRPXXXXXX.
Hi-C matrices used by the scripts in this repository are ICE-normalized sparse
matrices generated by [HiC-Pro](https://github.com/nservant/HiC-Pro).  To
regenerate them, run HiC-Pro with the appropriate reference genome (*Ascaris*
v50 or *Parascaris* v2) and apply ICE balancing.  Eigenvector files (derived
from Hi-C matrices using [FAN-C](https://fan-c.readthedocs.io/) `fanc
compartments`) are included in this repository under `data/eigenvectors/`.

## Repository structure

```
data/                           Shared reference files
├── 20000/                      HiC-Pro bin mappings (20 kb)
├── 40000/                      HiC-Pro bin mappings (40 kb)
├── interaction_regions/        Region pair BED files
├── eigenvectors/               EV1 files for Ascaris and Parascaris
│   ├── ascaris/
│   └── parascaris/
├── pu_v2_germ_to_soma_mapping.bed
├── pu_v2_eliminated.bed
├── AG_v50_eliminated_strict.bed
├── as_to_pu_chrom_order.txt
├── flip_or_not.txt
├── as_v50_prepde_to_postpde_coordinates.bed
├── AG_v50_chrom_sizes.txt
└── cbr_v50_500kb_windows_labeled.bed

interaction_scoring/            Interaction frequency quantification
triangle_heatmaps/              Triangular Hi-C heatmaps across development
eigenvector_comparison/         Ascaris vs Parascaris EV1 comparison
ascaris_eigenvector/            Ascaris EV1 across developmental stages
saddle_plots/                   Compartment saddle plots and strength analysis
```

Each analysis directory contains its own `README.md` with usage examples,
parameter descriptions, and a figure mapping table.
