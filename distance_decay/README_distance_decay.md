# distance_decay

Hi-C distance decay analysis for *Ascaris suum*.  Computes, combines,
normalizes, and plots expected contact frequency as a function of genomic
distance across developmental timepoints.

## Overview

`distance_decay.py` is a single script with three subcommands that allow
entering the pipeline at any stage:

- **`run`** â€” Full pipeline: runs [FAN-C](https://fan-c.readthedocs.io/)
  `fanc expected` on .hic files, then combines samples, re-bins, normalizes,
  and plots.  Requires FAN-C installed.
- **`combine`** â€” Starts from existing FAN-C output text files: combines,
  re-bins, normalizes, and plots.
- **`plot`** â€” Final step only: reads a pre-computed normalized text file and
  produces a publication-quality log-log distance decay plot.

### Normalization

Each sample is scaled so that its expected contact value in the first distance
bin equals the average first-bin value across all samples.  This corrects for
differences in total contact counts between libraries while preserving the
shape of each sample's decay curve.

Raw sequencing reads are available from SRA under accession SRPXXXXXX.  Hi-C
matrices (.hic format) are generated by HiC-Pro or cooler; expected values are
computed by FAN-C.

## Usage

### Plot from pre-computed normalized file (most common)

```bash
python distance_decay.py plot \
    --input data/combined_whole_genome_100000bp_normalized.txt \
    --output distance_decay \
    --title 'Normalized Whole Genome Expected Contacts' \
    --dpi 300
```

### Full pipeline from .hic files (requires FAN-C)

```bash
python distance_decay.py run \
    --chrom-sizes data/AG_v50_chrom_sizes.txt \
    --samples AG.v50.1cell.iced.5kb.hic AG.v50.48hr.iced.5kb.hic \
              AG.v50.60hr.iced.5kb.hic AG.v50.5day.iced.5kb.hic \
              AG.v50.10day.iced.5kb.hic \
    --output-binning 100000 \
    --outdir results/
```

### Combine existing FAN-C output files

```bash
python distance_decay.py combine \
    --pattern 'expected/AG.v50.*.iced.5kb/*genome.txt' \
    --output-binning 100000 \
    --outdir results/
```

## Parameters

### `plot` subcommand

| Flag | Default | Description |
|------|---------|-------------|
| `--input` | required | Normalized distance decay text file |
| `--output` | `distance_decay` | Output filename prefix |
| `--title` | `Normalized Whole Genome Expected Contacts` | Plot title |
| `--dpi` | 300 | Output resolution |
| `--width` / `--height` | 8 / 6 | Figure size in inches |

### `run` subcommand

| Flag | Default | Description |
|------|---------|-------------|
| `--chrom-sizes` | required | Chromosome sizes file |
| `--samples` | required | .hic files to process |
| `--output-binning` | required | Bin sizes in bp (e.g., 100000) |
| `--outdir` | `expected_combined` | Output directory |

### `combine` subcommand

| Flag | Default | Description |
|------|---------|-------------|
| `--pattern` | required | Glob pattern for FAN-C output files |
| `--output-binning` | required | Bin sizes in bp |
| `--outdir` | `combined_output` | Output directory |

## Outputs

- `{prefix}.svg` / `.png` / `.pdf` â€” distance decay plot
- `combined_{binsize}bp.txt` â€” binned combined table (combine/run modes)
- `combined_{binsize}bp_normalized.txt` â€” normalized table
- Per-chromosome normalized plots and tables (run mode)
